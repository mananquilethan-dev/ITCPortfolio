Portfolio 4: Computer Hardware & Software

Computer Hardware

Computer hardware refers to the physical components that make up a computer system. This includes input devices (keyboard, mouse), output devices (monitor, printer), storage devices (HDD, SSD), processing units (CPU, GPU), and networking hardware (routers, switches).

Types of Hardware

  • Input Devices: Keyboard, mouse, scanner
  • Output Devices: Monitor, printer, speakers
  • Processing Units: CPU, GPU
  • Storage Devices: HDD, SSD, external drives
  • Networking Devices: Routers, switches, modems

Examples in Institutions

  • Schools: Computers with projectors for teaching
  • Hospitals: Computers controlling MRI machines and storing patient records
  • Banks: ATMs and servers handling transactions
  • Government offices: Hardware for record management and digital services

Software and Its Types

Software is a set of instructions, programs, and data that tell a computer how to operate.

Types of Software

  • System Software: Operating systems like Windows, Linux, macOS
  • Application Software: MS Office, Adobe Photoshop, accounting software
  • Utility Software: Antivirus, disk cleanup, backup tools
  • Development Software: IDEs like Visual Studio, Eclipse

Examples in Institutions

  • Schools: Moodle, Google Classroom for digital education
  • Hospitals: Hospital Information Systems (HIS), Electronic Health Records (EHR)
  • Banks: Core banking software and transaction monitoring programs
  • Government offices: Payroll, tax, and citizen service management systems

Implementation in Institutions

  • Education: Online classes, digital libraries, student record management
  • Healthcare: Patient management, diagnosis, scheduling
  • Finance: Secure transactions, fraud detection, customer data management
  • Government: Tax collection, civil services, record maintenance

Analysis / Reaction

Before diving into this topic, I used to think that computers were just tools for typing or browsing the internet. I never realized how much planning and integration went into hardware and software before it could serve institutions. Learning about the physical components of computers, like CPUs, memory, and input/output devices, helped me see that every part has a specific function and limitation. Software is just as important. Together, they form the backbone of modern institutions, helping organizations function smoothly and enabling people to achieve more with technology. Personally, this motivated me to appreciate the thoughtfulness behind IT systems and inspired me to learn both hardware and software deeply.
